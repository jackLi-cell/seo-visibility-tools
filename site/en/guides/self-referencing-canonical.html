What Does Self-Referencing Canonical Mean

Each page's canonical should point to its own official URL, not all pointing to the homepage

What is Self-Referencing Canonical

Canonical 自指是指页面的 <link rel="canonical"> 标签指向自己的 URL。这明确告诉搜索引擎："这个 URL 是此内容的正式版本"。

Why Self-Referencing is Needed

  • Prevent URLs with parameters from being treated as separate pages
  • Clearly declare content ownership to avoid being flagged as duplicate
  • Consolidate link equity to the official URL
  • Handle HTTP/HTTPS, www/non-www variants

Common Errors

  • All pages canonical pointing to homepage (most common serious error)
  • Canonical using relative path instead of absolute URL
  • Canonical URL protocol inconsistent with actual URL
  • Canonical pointing to 404 page
  • Canonical pointing to URL in redirect chain
  • Dynamic page canonical contains session parameters

Correct Configuration Example

<!-- 正确：自指 -->
<link rel="canonical" href="https://example.com/blog/my-post">

<!-- 错误：指向Home -->
<link rel="canonical" href="https://example.com/">

<!-- 错误：相对路径 -->
<link rel="canonical" href="../blog/my-post">

Verification

Use Canonical Checker 验证你的 canonical 配置。
