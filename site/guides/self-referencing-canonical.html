Canonical 自指是什么意思

每个页面的 canonical 应该指向自己的正式 URL，不要全站指向首页

什么是 Canonical 自指

Canonical 自指是指页面的 <link rel="canonical"> 标签指向自己的 URL。这明确告诉搜索引擎："这个 URL 是此内容的正式版本"。

为什么需要自指

  • 防止带参数的 URL 被当作独立页面
  • 明确声明内容归属，避免被判定为重复
  • 统一链接权重到正式 URL
  • 处理 HTTP/HTTPS、www/non-www 的变体

常见错误

  • 所有页面 canonical 都指向首页（最常见的严重错误）
  • canonical 使用相对路径而非绝对 URL
  • canonical URL 与实际 URL 的协议不一致
  • canonical 指向 404 页面
  • canonical 指向重定向链中的 URL
  • 动态页面 canonical 包含会话参数

正确配置示例

<!-- 正确：自指 -->
<link rel="canonical" href="https://example.com/blog/my-post">

<!-- 错误：指向首页 -->
<link rel="canonical" href="https://example.com/">

<!-- 错误：相对路径 -->
<link rel="canonical" href="../blog/my-post">
